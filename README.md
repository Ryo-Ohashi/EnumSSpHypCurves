This repository provides the **Magma** implementation of the algorithm proposed in the following paper:

> [On superspecial hyperelliptic curves of Rosenhain forms](https://arxiv.org/abs/2606.10414) by Ryo Ohashi

### Example
If you would like to enumerate superspecial hyperelliptic curves of genus 4 in characteristic 23, use the following command:

```bash
magma g:=4 p:=23 main.m;
```
